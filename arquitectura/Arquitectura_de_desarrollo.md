# 🏗️ Arquitectura de Desarrollo

El desarrollo del proyecto consta de un proceso iterativo, en el cual nos centraremos en el desarrollo de las funcionalidades del programa. Este mismo constará de **4 fases**:

## 🔹 Fase 1
En esta fase lo que buscamos es la integración de las **direcciones IP ya conocidas** por el sistema.

## 🔹 Fase 2
Se realiza un **sondeo de las conexiones** para visualizar las conexiones conocidas.

## 🔹 Fase 3
En caso de una detección, se realiza la **notificación al área correspondiente**.

## 🔹 Fase 4
Se realiza la **mejora del sistema**.
