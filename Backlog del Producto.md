# Historias de Usuario (Priorizadas)

## ALTA

1. **Como analista de seguridad, quiero recibir alertas en tiempo real de intentos de intrusión para responder rápidamente a amenazas activas.**  
   - *Criterio de aceptación:* El sistema debe notificar en menos de 30 segundos desde la detección del intento de intrusión.

2. **Como administrador de seguridad, necesito un panel que muestre el estado actual de la red para identificar anomalías de forma visual.**  
   - *Criterio de aceptación:* El panel debe actualizarse automáticamente cada minuto y mostrar métricas clave de seguridad.

3. **Como responsable de TI, quiero que el sistema ejecute respuestas automáticas predefinidas ante amenazas conocidas para minimizar el tiempo de exposición.**  
   - *Criterio de aceptación:* El sistema debe ejecutar respuestas en menos de 1 minuto tras la detección.

## MEDIA

4. **Como analista de seguridad, quiero programar escaneos de vulnerabilidades para detectar puntos débiles antes que los atacantes.**  
   - *Criterio de aceptación:* Los escaneos deben ser configurables por tipo de activo y frecuencia.

5. **Como gerente de seguridad, necesito informes periódicos de incidentes para evaluar tendencias y patrones de ataque.**  
   - *Criterio de aceptación:* Los informes deben incluir estadísticas, gráficos y recomendaciones.


## BAJA

6. **Como analista, quiero integrar fuentes de inteligencia de amenazas externas para mejorar la detección de amenazas emergentes.**  
   - *Criterio de aceptación:* El sistema debe actualizar automáticamente los indicadores de compromiso desde fuentes configurables.

7. **Como administrador, necesito gestionar permisos de usuarios del sistema para limitar el acceso según roles y responsabilidades.**  
   - *Criterio de aceptación:* Debe existir un sistema de roles con permisos granulares configurables.
